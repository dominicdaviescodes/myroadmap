As you know, the web development world is fast changing so one quick update that has happened since the next video was filmed is that the property you are going to learn about: grid-gap has been changed to gap

Wrap all the items in a container. style it display: grid. Then set grid-template-columns, tell page how many columns. If use 2 values like 300px 300px it will put elements into 2 columns. to make a gap between elements gap: 20px;

because we set the grid-template-columns statically to 300px each it's not responsive.

fr syntax

If you use 1fr, the page will use 1 column.

If you use 1fr 1fr, the page will use 2 columns.

If you use 1fr 1fr 2fr, the page will use 2 columns of 1fr and 1 col of 2fr.

grid-template-rows: 1fr 2fr; the 2nd row is twice the height of the 1st row.

we can use shorthand. grid-template-columns: 1fr 1fr 1fr; we could say repeat(3, 1fr)

grid-template-columns: auto; auto property fits the content.

A common grid setting

grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
