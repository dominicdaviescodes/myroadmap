Transitions

Transitions enable you to define the transition between two states of an element. Different states may be defined using pseudo-classes like :hover or :active or dynamically set using JavaScript.

This example without transition simply switches instantly from square to circle.

we can style the changeover delaying it slightly and making the transition smoother or ease in or dozens of other effects.


shape 2. I set a transition: .5s;

everything changes. You can specify all, so all properties will change.

In JS you can add and remove classes, it's not just hovering.

The transition CSS property is a shorthand property for transition-property, transition-duration, transition-timing-function, and transition-delay.


shape 3. transition: background-color 3s;

the shape change is instant, however the bg color changes slowly. you could even add a delay in seconds and the transition won't begin until then.


There are so many ease in options Easing functions specify the rate of change of a parameter over time.
